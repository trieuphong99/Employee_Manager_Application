### Install project

1. git clone git@gitlab.com:Bunbu/imp.git
2. cd imp
3. gem install bundler
4. bundle install
5. bundle exec rake db:create
6. bundle exec rake db:migrate
7. bundle exec rake db:seed
8. rails s (khởi động server)
9. truy cập: localhost:3000
